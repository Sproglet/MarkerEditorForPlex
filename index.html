Welcome to the Plex Intro Editor. To begin, select a TV library from the dropdown below (if one is not already selected) and search for the show that you want to edit the intro markers for. After selecting a show and a season, click on an episode name to show the marker table (or Ctrl+click to show/hide markers for all episodes in the season). The table will allow you to add, edit, and delete its markers. Note that multiple markers can also be added for an episode, and all will trigger within Plex (tested on web and desktop clients), so can also be used to add credit skips.
